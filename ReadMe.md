#### FindarAssessment


#### TECHNOLOGY
- **Java**
- **Maven**
- **Docker**


#### REQUIREMENT
- [Docker](https://docs.docker.com/engine/install/)

#### Run The Application
Execute the commands:

```bash
    ./mvnw clean install     
    docker-compose build    
    docker-compose up     
```

#### Run The Tests
```bash
    ./mvnw test              
```
